    [BapuRo]新人简易攻略

    2020-4-12 22:46| 发布者: BapuRo管理员| 查看: 197| 评论: 0

    [BapuRo]新手简易攻略：

    n  账号注册

    n  建人物

    n  进入叭噗大陆

    n  领取新手装备

    n  购买萌新武器

    n  练级（推荐以人2作为练级主战场，练级的同时可以做掉一套重要装备。）

    PS：服务器支持混转，建议使用枪手进行转生，转生后再用自己需要的职业

    职业选择，开荒推荐

    n  武术宗师

    n  神射手

    n  骑士领主

    n  神官

    n  炼金术士

    PS:服务器有平转功能，待装备入门之后再进行职业变更。

     

    装备制作顺序推荐：

    Step1

    n  一星影子装备，材料来源：生命体研究所二楼

    PS:完成影子装备之后根据自身需求加点，属性会有一定提升

    Step2

    n  普通时装化卡片，材料来源：对应卡片怪物掉落合成

    PS:卡片满足开荒需求，进一步的提升自己

    Step3

    n  一星普通装备，材料来源：生命体研究所三楼

    PS:在人3制作一星装备的同时，还可以积攒人3卡片制作时装化人3卡片。

     

    至此开荒装备制作完成。

    完成一星普通装备可以完成成就并获得称号：赏金初心者

    拥有该称号之后，可以在NPC处接取每日赏金任务，获得金钱以及道具。

    接下来推荐根据自身情况选择主玩职业，对照着各职业的宝石组合模式

    开始搜罗宝石

    宝石主要来源有：野外瞬刷BOSS，非瞬刷BOSS，初级道场，远古生物副本，每日赏金任务

    在游戏初期，宝石的形状需求大于宝石类型的需求

    组合出效果是游戏初期的重中之重

    推荐先完成组合，再慢慢调整宝石的类型。
